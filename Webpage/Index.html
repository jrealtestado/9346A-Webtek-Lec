HTML & HTTP Notes

  • World Wide Web(1989, Tim Berners-Lee)system of internet that allows documents to be connected to other documents with hypertext links.
  • Hyper-Text Markup Language known as HTML . Markup is a way of document to have designated meaning and at is a language that is used to create or design a web page.
  • HTML is focused on structure and content while the CSS handle the presentation or aesthetic and JavaScript handles the behavioral aspects of the site.HTML was created by Tim Berners-Lee and W3C is responsible for evolving HTML
  • Email – earliest application that uses the internet
  • Internet (1969) – connection among others
  • Nodes- usually computers or devices
  • Interconnection Technology – used to connect nodes
    • Wired – have actual physical connection
    • Wireless - don’t have actual physical connection
  • Protocols – standard way of doing things
  • Tranceiver – device that receives and transmits data
  • Gopher – indexes servers of information
  • Usenet – similar to bulletin board
  • HTTP – port 80 (default)
  • SSL – port 443

Abbreviations:

  • HTTP - HyperText Transfer Protocol
  • HTML - HyperText Markup Language
  • WAIS (Wide Area Information Servers) – databases in servers
  • LAN (Local Area Network) – network for a small geographic area
  • ISP (Internet Service Provider) – connects LAN to LAN via gateway
  • SSL – Secure Socket Layer
  • TLS – Transport Layer Security

Packet Switched vs Circuit Switched:

  • Packet Switch – store and forward; connection to node don’t need connection to that node
  • Circuit Switch – connection should be maintained

Servers:

  • Origin Server
  • Proxy Servers, gateways, tunnels (encryption)

HTTP Version History:

  • HTTP 0.9 (1991)
  • HTTP 1.0 (RFC 1945, May 1996)
  • HTTP 1.1 (RFC 2068 Jan 1997, RFC 2616 Jun 1999), RFC 7230-7235 (Jun 2014)
  • HTTP 2 (RFC 7540 May 2015)

HTTP Status Codes

Informational (1xx):

  • 100 Continue
  • 101 Switching Protocols

Success (2xx):

  • 200 OK
  • 201 Created
  • 202 Accepted
  • 203 Non-Authoritative Information
  • 204 No Content
  • 205 Reset Content
  • 206 Partial Content

Redirection (3xx):

  • 300 Multiple Choices
  • 301 Moved Permanently
  • 302 Found
  • 303 See Other
  • 304 Not Modified
  • 305 Use Proxy
  • 306 Switch Proxy
  • 307 Temporary Redirect

Client Error (4xx):

  • 400 Bad Request
  • 401 Unauthorized
  • 402 Payment Required
  • 403 Forbidden
  • 404 Not Found
  • 405 Method not Allowed
  • 406 Not Acceptable
  • 407 Proxy Authentication Required
  • 408 Request Timeout
  • 409 Conflict
  • 410 Gone
  • 411 Length Required
  • 412 Precondition Failed
  • 413 Request Entity Too Large
  • 414 Request-URI Too Long
  • 415 Unsupported Media Type
  • 416 Request Range Not Satisfiable
  • 417 Expectation Failed

Server Error (5xx):

  • 500 Internal Server Error
  • 501 Not Implemented
  • 502 Bad Gateway
  • 503 Service Unavailable
  • 504 Gateway Timeout
  • 505 HTTP Version Not Supported
