HTML & HTTP Notes

  • World Wide Web(1989, Tim Berners-Lee) – system of internet that allows documents to be connected to other documents with hypertext links.
  • Email – earliest application that uses the internet
  • Internet (1969) – connection among others
  • Nodes- usually computers or devices
  • Interconnection Technology – used to connect nodes
    • Wired – have actual physical connection
    • Wireless - don’t have actual physical connection
  • Protocols – standard way of doing things
  • Tranceiver – device that receives and transmits data
  • Gopher – indexes servers of information
  • Usenet – similar to bulletin board
  • HTTP – port 80 (default)
  • SSL – port 443

Abbreviations:

  • HTTP - HyperText Transfer Protocol
  • HTML - HyperText Markup Language
  • WAIS (Wide Area Information Servers) – databases in servers
  • LAN (Local Area Network) – network for a small geographic area
  • ISP (Internet Service Provider) – connects LAN to LAN via gateway
  • SSL – Secure Socket Layer
  • TLS – Transport Layer Security

Packet Switched vs Circuit Switched:

  • Packet Switch – store and forward; connection to node don’t need connection to that node
  • Circuit Switch – connection should be maintained

Servers:

  • Origin Server
  • Proxy Servers, gateways, tunnels (encryption)

HTTP Version History:

  • HTTP 0.9 (1991)
  • HTTP 1.0 (RFC 1945, May 1996)
  • HTTP 1.1 (RFC 2068 Jan 1997, RFC 2616 Jun 1999), RFC 7230-7235 (Jun 2014)
  • HTTP 2 (RFC 7540 May 2015)
